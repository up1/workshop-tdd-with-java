# workshop-tdd-with-java

## Testing
* Unit
* Integration
* Contract
* End-to-End

## Start server
```
$mvnw spring-boot:run
```

Open url=http://localhost:8080 in browser

## Build and testing project 

```
$mvnw clean test
```

## Code coverage

```
$mvnw clean package
```

Open file ./target/site/jacoco/index.html in browser
